# ClipboardPurifier

## 简介

ClipboardPurifier 是一个轻量级的 Windows 剪贴板管理工具。它旨在在后台运行，自动清理剪贴板内容，移除可能不需要的格式或数据，例如 HTML 标记、冗余空格等，从而提供纯文本或更简洁的剪贴板体验。它通过系统托盘图标提供简单的用户界面和控制功能。

## 手动编译

### 环境要求

* **MSYS2 with UCRT64:** 本项目推荐使用 MSYS2 UCRT64 终端环境进行编译。请确保您已安装 MSYS2，并且在 UCRT64 终端中操作。

### 编译步骤

1. **打开 UCRT64 终端:**

   在项目文件目录下启动 `MSYS2 UCRT64` 终端。


2. **执行 make 命令:**

   在项目目录下，只需运行 `make` 命令即可一键编译。在此之前，需要安装gtk4的相关工具和库，教程见 [安装GTK4-官方教程](https://www.gtk.org/docs/installations/windows/)

   ```bash
   make
   ```

   `make` 命令将自动处理所有依赖项的编译和链接，生成可执行文件。


3. **程序运行库依赖:**

   编译成功后，您可以在项目目录中找到生成的可执行文件 `ClipboardPurifier.exe` 。双击运行。

> 对于手动编译的程序，由于程序采用动态链接库，建议直接将gtk4下所有lib文件复制到与可执行程序相同的目录下

### 注意事项

* 如果在编译过程中遇到任何依赖错误，请确保您的 MSYS2 UCRT64 环境已通过 `pacman` 命令安装了所有必要的开发工具和库（例如 `make`, `gcc`, `g++` 等）。
* 本程序主要设计用于 Windows 操作系统。
